# 만 14세 미만 원격 학습 기록 출시 기준

확인일: 2026-07-21

## 공식 근거

- 국가법령정보센터, 개인정보 보호법 제22조의2(아동의 개인정보 보호):
  <https://www.law.go.kr/LSW/lsSideInfoP.do?docCls=jo&joBrNo=02&joNo=0022&lsiSeq=270351&urlMode=lsScJoRltInfoR>

현재 조회된 법령은 `[시행 2025. 10. 2.] [법률 제20897호, 2025. 4. 1.,
일부개정]`이다. 제22조의2는 만 14세 미만 아동의 개인정보를 처리하기
위해 법에 따른 동의를 받아야 할 때 법정대리인의 동의와 동의 여부 확인을
요구한다. 동의를 받기 위한 최소 정보는 예외적으로 먼저 받을 수 있고,
아동에게는 이해하기 쉬운 형식과 언어로 고지해야 한다.

## Math Assist 적용

- 계정 없는 공개 학습과 기기 localStorage 저장은 계속 기본 경로다.
- 어린이는 `기록 저장하기`를 직접 선택할 수 있지만, 만 14세 미만의 원격
  프로필 생성과 동기화는 법정대리인 동의 확인 뒤에만 완료 상태가 된다.
- 학습 번호, 4자리 PIN, 진도 데이터가 실명·이메일을 포함하지 않더라도
  서버가 특정 학습자에게 기록을 지속적으로 연결하므로, 단순히 익명이라고
  가정해 동의 게이트를 생략하지 않는다.
- 동의 전에는 원격 저장 성공으로 표시하거나 로컬 기록을 삭제하지 않는다.
- 아동 고지 문구는 수집 항목, 목적, 보유 기간, 삭제·복구 방법을 어린이가
  이해할 수 있는 짧고 명확한 언어로 제공한다.

## 공개 배포 차단 조건

다음이 모두 해결되기 전 원격 계정 기능은 비공개 staging을 넘지 않는다.

1. 적용 가능한 적법 근거와 법정대리인 동의·확인 방식의 개인정보 검토
2. 계정 삭제 후 보존·복구 유예와 영구 삭제 시점
3. 보호자 연결 해제 즉시 권한·복구 수단을 철회하는 방식
4. 로그인 세션의 절대·유휴 만료와 전체 세션 철회 방식
5. 개인정보 처리방침, 아동용 고지, 동의 증적, 접근·삭제 요청 절차

이 문서는 법률 자문을 대체하지 않는다. 구현과 보안 테스트를 시작할 수 있는
제품 게이트이며 공개 배포 전에는 별도 개인정보 검토 결과를 기록해야 한다.
