# 랜딩·게스트 학습 홈 핸드오프

Status: implemented and verified locally

## What changed

- `/`를 가입 없이 시작하는 간결한 랜딩으로 교체했다.
- `/home`에 최초 학년 선택, 학년 변경, 최근 학습 복귀, 공통 진도 요약을 추가했다.
- 기존 5학년 단원 목록을 `/grade/5`로 옮겼다.
- 학습자 화면의 `홈으로` 링크를 `/home`으로 정리하고 문제 검수 보드를 랜딩에서 제거했다.
- `src/lib/guest-home.ts`가 기존 학년별 localStorage 기록을 읽어 공통 홈 상태로 변환한다.
- 1학년 재진입 시 첫 미션이 아니라 복습 또는 다음 미션을 열도록 복원 동작을 바로잡았다.

## What is still risky

- 기록은 한 브라우저에만 남으며 브라우저 데이터 삭제나 다른 기기 이동을 복구하지 못한다.
- 홈 어댑터는 기존 저장 키와 진행 스키마를 읽으므로 해당 계약이 바뀌면 함께 갱신해야 한다.
- 로그인 도입 전에는 여러 아이가 같은 브라우저에서 별도 프로필을 사용할 수 없다.

## What the next agent should do

- 학습자 프로필을 추가할 때 홈 UI를 다시 만들지 말고 `guest-home`의 데이터 제공 경계를 교체한다.
- 계정 기능보다 먼저 로컬 기본 프로필과 기존 키 마이그레이션 방식을 설계한다.
- 학년별 진행 스키마를 바꿀 때 최초 방문, 재방문, 5학년 세션 복귀 테스트를 다시 실행한다.
