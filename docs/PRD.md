# Product Requirements

This file is a harness-friendly entry point. The detailed product requirements remain
in `math_prd.md` at the repository root.

## Goal

Provide Korean elementary math learners with concept explanations, short practice
sets, deterministic grading, and step-by-step solution feedback.

## Users

- Elementary students practicing math concepts on a tablet.
- Parents or tutors who want predictable practice and clear correction feedback.

## Core Experience

1. Select a unit and concept.
2. Read a concise explanation.
3. Complete a 10-question practice session.
4. Submit answers for deterministic grading.
5. Review correctness and step-by-step solutions.

## Non-Goals

- No commercial production operations are assumed.
- No database-backed admin workflow.
- No AI-based grading or partial-credit scoring.

## Reference

See `math_prd.md` for schemas, endpoints, detailed flows, and content requirements.
