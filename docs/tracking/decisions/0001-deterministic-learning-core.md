# 규칙 기반 생성과 채점

## 상황

초등 수학 앱은 자연어 설명이 친절해야 하지만 계산 실수나 답의 흔들림은 학습 신뢰를 바로 무너뜨린다. 같은 문제를 다시 풀 때 정답과 풀이 단계가 달라져서도 안 된다.

## 결정

문제 매개변수 생성, 정답 계산, 답 정규화, 정오 판정, 원본 풀이 단계는 규칙과 검증된 템플릿으로 만든다. AI가 추가되더라도 자연어 힌트와 표현 개선에만 쓰며 기존 정답·채점·풀이를 바꿀 권한을 주지 않는다. 점수는 정답 또는 오답 두 결과만 사용한다.

## 선택하지 않은 안

- AI가 문제와 정답을 즉석 생성하고 채점하는 방식은 계산 오류와 재현 불가능한 판정이 학생에게 직접 노출되므로 선택하지 않았다.
- 교사가 결과를 수동 검토하는 방식은 개인용 즉시 학습 흐름에서 매 문제의 빠른 피드백을 제공할 수 없어 선택하지 않았다.

## 결과

새 수학 유형은 코드·데이터 규칙과 테스트를 추가해야 하므로 콘텐츠 제작 비용이 더 든다. 대신 정답을 재현하고 실패 사례를 회귀 테스트로 고정할 수 있다. 자유 서술형 의미 채점과 부분 점수는 현재 구조에서 제공할 수 없다.
