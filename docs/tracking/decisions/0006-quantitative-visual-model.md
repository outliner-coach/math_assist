# 정량 그림의 단일 수학 모델

## 상황

고정 SVG가 문제와 비슷하게 보이더라도 실제 길이·넓이·겹침 관계가 문제 수치와 다르면 그림을 믿은 학습자가 틀리게 된다. 세 도형 겹침 문제에서 실제로 수치상 0인 영역이 그림에는 존재하는 문제가 발견됐다.

## 결정

그림을 장식·개략·정량으로 분류하고, 정량 그림은 문제 수치·정답·영역 위상·비율·라벨을 하나의 수학 모델에서 파생한다. 정량 그림은 위상과 핵심 대소 관계를 전부 보존하고 비율 오차 ±10%, 라벨 영역 포함, 0 영역 비표시를 자동 검증한다. 답 전용 값은 제출 또는 풀이 공개 전까지 DOM에 만들지 않는다.

## 선택하지 않은 안

- 디자이너가 고정 좌표 SVG를 눈으로 비슷하게 조정하는 방식은 생성되는 모든 수치 조합의 위상과 비율을 보장할 수 없어 선택하지 않았다.
- 그림을 모두 장식으로 낮추는 방식은 도형과 응용 문제에서 시각 정보를 이용해 생각하게 하려는 학습 목표를 잃으므로 선택하지 않았다.

## 결과

정량 그림 추가에는 수학 모델과 렌더러뿐 아니라 위상·비율·라벨 테스트가 필요하다. 기존 세 도형 겹침 렌더러는 이 결정을 만족하지 못해 교체하거나 수정 전 노출을 중지해야 한다.
