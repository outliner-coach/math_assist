# 손실과 벌점 없는 보상

## 상황

짧은 미션과 보상은 어린이의 다음 행동을 돕지만, 연속 기록 손실·힌트 벌점·리더보드는 도움을 요청하거나 쉬는 일을 실패처럼 느끼게 할 수 있다. 같은 고정 문제를 반복해 보상만 얻는 경로도 막아야 한다.

## 결정

보상은 새 구체 문제 해결, 힌트 없는 첫 시도, 예정된 복습처럼 생산적인 행동에 더한다. 힌트를 사용하거나 하루 쉬었다고 기존 경험치를 깎지 않는다. 같은 문장·정답·보기·그림의 문제는 시드가 달라도 한 번만 완료 보상을 주며, 캐릭터와 미션은 학습 코어의 결과를 표현할 뿐 채점을 통제하지 않는다.

## 선택하지 않은 안

- 힌트 사용 시 점수나 경험치를 깎는 방식은 필요한 도움을 피하게 만들 수 있어 선택하지 않았다.
- 매일 접속하지 않으면 연속 기록을 즉시 0으로 만드는 방식은 어린이의 생활 리듬을 벌주는 효과가 있어 선택하지 않았다.
- 시드만 다르면 새 보상을 주는 방식은 고정 데이터 문제가 반복될 때 경험치를 무한히 얻을 수 있어 선택하지 않았다.

## 결과

구체 문제의 내용 지문을 안정적으로 유지해야 하며 문제 문장·정답·보기·그림을 바꾸면 보상 중복 판단도 영향을 받는다. 일일 경계는 기기 현지 자정으로 통일해야 한다. 복수 화폐, 상점, 리더보드, 보스전은 현재 제품 방향에서 제외된다.
