# 완성되지 않은 입력을 오답과 분리

## 상황

2·3학년 구조화 입력은 단위나 값 형식이 완성되지 않으면 다시 입력하게 하지만, 5학년 숫자 입력은 `1/`, `-`, `.`처럼 비어 있지 않은 값도 확인 즉시 오답으로 확정하고 답을 잠근다. 계산 실수와 입력을 덜 마친 상태가 같은 학습 결과가 되는 문제가 있다.

## 결정

숫자와 구조화 입력은 정규화에 성공한 뒤에만 정오를 판정한다. 완성되지 않은 형식은 답을 잠그거나 오답으로 기록하지 않고, 고쳐야 할 형식을 구체적으로 안내한다. 이 규칙을 모든 학년에 동일하게 적용한다.

## 선택하지 않은 안

- 비어 있지 않은 모든 값을 즉시 채점하는 방식은 구현이 단순하지만 입력 중 실수를 수학적 오답으로 남기고 어린이가 같은 문제에서 바로 고칠 기회를 없애므로 선택하지 않았다.
- 잘못된 형식을 자동으로 추측해 보정하는 방식은 사용자가 의도하지 않은 수를 채점할 수 있으므로 선택하지 않았다.

## 결과

5학년 연습 화면은 채점 전에 정규화 성공 여부를 검사해야 한다. 형식 오류는 세션의 확인 상태, 오답 수, 진도, 보상을 바꾸지 않아야 한다. 답이 유효해진 뒤에는 기존처럼 정답 또는 오답 두 결과만 사용하며 부분 점수는 생기지 않는다.
