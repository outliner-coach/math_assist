# 현재 상태

기준일: 2026-07-21

## 구현되고 검증된 범위

- 공개 랜딩, 기기 로컬 학습 홈, 1·2·3·5학년 진입 경로가 구현되어 있다. 홈은 기존 학년별 저장 이름을 바꾸지 않고 이어하기를 계산한다.
- 1학년은 96개 미션, 2학년은 12개 단원·144개 미션, 3학년은 36개 미션을 제공한다. 1·2학년은 날짜·재도전 시드, 내용 기반 중복 보상 방지, 경험치·레벨·오늘 목표·복습·숙련을 제공한다.
- 5학년은 12개 단원, 22개 개념, 개념당 A·B·C 10문제 세트를 제공한다. 숫자·분수·대분수 정규화, 문항별 즉시 확인, 오답 재도전, 2시간 세션 복구, 개념별 진도가 구현되어 있다.
- 도형 문제는 React SVG를 사용하며 다수의 답 전용 치수와 결과값이 확인 전 숨겨지는 회귀 테스트가 있다.
- 풀이장은 5학년 연습에 제공되고 iPadOS의 선택 제스처·포인터 캡처 실패·다중 포인터 경로가 집중 테스트로 보호된다. 문제별 획 복구와 되돌리기는 아직 없다.
- 로그인, 서버 저장, 원격 AI 호출은 현재 구현되어 있지 않다. 학습 기록은 브라우저 localStorage에만 남는다.

## 남은 일

우선순위와 선행 관계 순서다.

1. `three_shape_overlap` 문제의 수치와 고정 SVG 위상이 일치하지 않는 신뢰 오류를 차단한다. 격자 기반 영역 모델과 위상·비율·라벨 테스트가 준비될 때까지 해당 템플릿을 세션에서 제외하는 방안도 포함한다.
2. 5학년 숫자 입력에서 완성되지 않은 형식을 확인해도 오답으로 기록하거나 답을 잠그지 않도록 형식 검사를 분리하고 집중 테스트를 추가한다.
3. 숫자 난이도와 `knowing / applying / reasoning` 인지 유형을 분리하는 문제 청사진 필드와 검증기를 추가하고, 기존 5학년 660개 템플릿의 문제군·표현·추론 공백을 감사한다.
4. 학년별 세션과 네 종류 진도를 공통 학습 활동·시도 기록·진도 저장 인터페이스로 읽는 경계를 만든다. 기존 localStorage 기록의 무손실 회귀 테스트가 선행되어야 한다.
5. 전 학년 풀이장에 되돌리기와 문제별 임시 복구를 추가하고, 학년군별 문제 수·글 밀도·캐릭터·도구·보상 노출을 분리한다.
6. 게스트 기록의 선택적 원격 저장을 구현한다. 자동 생성 학습 번호, 4자리 숫자 비밀번호, 로그인 유지, 속도 제한, 일회성 복구 카드, 선택적 보호자 연결을 포함한다.
7. 4학년과 6학년 콘텐츠를 교육과정·문제군·표현 검증과 함께 추가한다. 4학년은 현재 홈에서 준비 중 상태만 보여 준다.
8. 자연어 힌트 품질 향상이 실제로 필요할 때만 원격 AI를 추가한다. AI는 정답·채점·원본 풀이를 결정하지 않는다.

## 막힌 결정

- 원격 계정의 삭제 후 보존 기간, 보호자 연결 해제 시 처리, 로그인 세션 만료 시간은 정해지지 않았다. 이 결정이 내려지기 전에는 계정 기능을 실제 배포할 수 없다.
- 세 도형 겹침 정량 그림은 현재 수학 데이터와 렌더 위상이 어긋난다. 제품 코드와 콘텐츠를 함께 바꾸고 테스트해야 하므로 문서 정리만으로 해결할 수 없다.
